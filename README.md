## 1) What is the difference between null and undefined?

     null = কোনো মান নেই
     undefined = ভেরিয়েবল বানানো হয়েছে কিন্তু মান দেওয়া হয়নি।

## 2) What is the use of the map() function in JavaScript? How is it different from forEach()?

     map() = অ্যারে থেকে নতুন অ্যারে বানায়
     forEach() = শুধু লুপ করে, নতুন অ্যারে দেয় না

## 3) What is the difference between == and ===?

     == -> শুধু মান চেক করে
     === -> মান ও টাইপ দুইটাই চেক করে

## 4) What is the significance of async/await in fetching API data?

     async = ফাংশন প্রমিস রিটার্ন করে
     await = প্রমিস শেষ হওয়া পর্যন্ত অপেক্ষা করে

## 5) Explain the concept of Scope in JavaScript (Global, Function, Block).

     Global = সব জায়গা থেকে ব্যবহার করা যায়
     Function = শুধু ফাংশনের ভিতরে ব্যবহার
     Block = শুধু {} এর ভিতরে ব্যবহার
